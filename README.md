# facebook-shame-gate
Ever open Facebook without even realizing it? This Chrome extension 
catches you before you fall into the scroll trap. It shows a warning 
page with a countdown timer every time you try to visit Facebook — 
giving you 10 seconds to reconsider. Click to skip or just wait it out.

## How to install
1. Download/clone this repo
2. Go to chrome://extensions
3. Enable Developer Mode
4. Click "Load unpacked" and select this folder
